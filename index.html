Getting Started

tonic-site-generator can be retrieved using npm.

npm

~ $ npm install -g tonic-site-generator

Quick-start

For the impatient, here's how to get boilerplate tonic-site-generator up and running. 
~ $ git clone git@github.com:Kitware/tonic-site-generator/.git
~ $ cd tonic-site-generator
~/tonic-site-generator $ npm install 
~/tonic-site-generator $ npm run test:basic
# => Now browse to http://localhost:3000
